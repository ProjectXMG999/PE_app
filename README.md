# PE_app

A web application deployed on Netlify.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/ProjectXMG999/PE_app.git
cd PE_app
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

This project is deployed automatically via [Netlify](https://www.netlify.com).
